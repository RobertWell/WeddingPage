# 3D 報表視覺強化建議

## 現況觀察
- 目前已經使用陰影（`--shadow-*`）、漸層底色、grain 紋理、leaf pattern、hero overlay 來營造深度與層次。
- 視覺「立體感」主要來自卡片陰影與背景漸層，但裝飾元素仍偏平面、重複度高。

## 可增加的「非配色」變化方向

### 1) 多層裝飾結構（替換葉片裝飾）
- **新增第二層裝飾**：在 `.leaf-pattern::after` 增加「幾何/花瓣/細線波紋」SVG，讓裝飾有前後層次。
- **切換裝飾主題**：把 `--leaf-pattern` 改為不同主題（例如藤蔓、花瓣、弧線、星點），避免只有改顏色。
- **裝飾區塊化**：在重要區塊（邀請卡 / RSVP）放置固定裝飾角（corner ornaments），而不是只在背景鋪滿。

### 2) 層次化的光影與浮起效果
- **雙陰影堆疊**：卡片陰影可使用兩層 shadow（柔霧 + 硬邊），提高 3D 層次。
- **浮起 Hover**：卡片或按鈕 hover 時加上 `translateY(-2px)` + `box-shadow` 增強互動感。
- **光暈層**：在卡片背後加一個模糊光暈 `::before`（低透明度）形成發光底座。

### 3) 建築式的背景層（不只是純色或漸層）
- **淡色光暈 Blob**：用 2–3 個大型圓形漸層元素（低透明度）在頁面底層做柔焦背景，避免背景只有平滑漸層。
- **斜切切片**：在 section 頂/底加入斜切或弧形分隔（SVG 或 pseudo-element），改變版面結構。
- **微視差**：讓背景裝飾與內容在 scroll 時微幅不同速率（1–3px），提升立體感。

### 4) SVG 裝飾的質感提升
- **描邊漸層**：SVG 改用 `stroke` 漸層 + `stroke-dasharray`，營造手繪筆觸感。
- **加上內陰影/高光**：對 SVG 元素加 `filter: drop-shadow(...)`，創造浮起效果。
- **局部填色**：用 2–3 個重點色塊填入局部，形成「繪本式」的層次。

### 5) 建立「裝飾主題套件」
- **主題 A：森系雕花**（葉片 + 細藤 + 花邊）
- **主題 B：幾何紙雕**（菱形 + 斜線 + 層疊紙張陰影）
- **主題 C：夢幻泡泡**（柔光圓點 + 淡色光暈 + 漸層氣泡）
- **主題 D：金箔點綴**（細碎點 + 噴墨質感 + 金屬光澤）

### 6) 文字區塊立體處理
- **標題陰影**：標題加非常細微的 shadow，讓字浮出背景。
- **描邊字**：主標題可加 1px 輕描邊（透明度 10–20%），增強層次。
- **背景字塊**：標題後方增加小色塊或半透明紙片，形成「貼紙感」。

## 三種可落地的變更方向（更清楚的選擇）
> 你可以挑其中一個方向做「全站一致」，或是分別用在不同區塊。

1. **裝飾形態升級（Decor Upgrade）**
   - 重點：改變裝飾的“形狀與結構”，不是只換顏色。
   - 作法：在 `.leaf-pattern` 疊一層新 SVG（藤蔓 / 幾何 / 氣泡 / 金箔點），並在卡片角落放固定角飾。
   - 效果：一眼就知道風格不同、主題更鮮明。

2. **結構層次升級（Layer Upgrade）**
   - 重點：讓背景與區塊的“層”更清楚。
   - 作法：section 之間用弧形或斜切分隔；加一兩個大型柔焦 blob 做底層。
   - 效果：畫面有前中後景，深度感更強。

3. **互動與浮起升級（Motion Upgrade）**
   - 重點：讓元素“動起來”而不花俏。
   - 作法：卡片/按鈕 hover 加微位移 + 雙陰影；SVG 加輕微漂浮動畫。
   - 效果：互動更精緻，3D 感提升。

## 最推薦的快速改動組合（低成本高變化）
1. **為 `.leaf-pattern` 加第二層 SVG 裝飾 + 降低不透明度** → 立即有前後層次。
2. **卡片加入 hover 浮起與雙陰影** → 增強 3D 互動感。
3. **section 分隔加弧形 SVG** → 改變整體版面結構，不只是換色。

## 下一步建議（清楚的行動路線）
1. **先選 1 個主題套件（A/B/C/D）**，定義該主題的裝飾形狀與細節。
2. **對應 3 個層級做改動**：
   - 背景層：更換 `--leaf-pattern` / 新增底層 blob。
   - 區塊層：加 section 分隔（弧形 / 斜切）。
   - 元件層：卡片 hover + 角飾。
3. **從邀請卡 + RSVP 區塊開始**，範圍小但視覺回饋最明顯。
